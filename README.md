# Netflix-Data-Analysis
Exploratory data analysis project on a  Netflix dataset. This project explores and analyzes the Netflix content catalog to uncover trends and patterns across different dimensions using Python, pandas, and Plotly.

## Technology Used:
### Language:

- Python (Pandas, NumPy, TextBlob)
- Plotly Express & Graph Objects for interactive charts
- Jupyter Notebook for analysis
- GitHub Pages for web-hosted interactive charts

### IDE:
Jupyter Notebook

## Dataset:
**Source**: [Kaggle: Netflix Movies And TV Shows](https://www.kaggle.com/datasets/shivamb/netflix-shows?resource=download) <br>
**Entries**: ~8,800+
- **Features include**: title, director, cast, country, date added, release year, rating, duration, genre, description, etc.
